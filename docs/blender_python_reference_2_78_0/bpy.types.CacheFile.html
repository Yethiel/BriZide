Path to external displacements file

Type:string, default “”, (never None)

Type:enum in [‘POS_X’, ‘POS_Y’, ‘POS_Z’, ‘NEG_X’, ‘NEG_Y’, ‘NEG_Z’], default ‘POS_X’

Type:float in [-500000, 500000], default 0.0
