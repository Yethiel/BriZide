class bpy.types.ColorRamp(bpy_struct)¶

Set color mode to use for interpolation

Type:enum in [‘RGB’, ‘HSV’, ‘HSL’], default ‘RGB’
elements¶

hue_interpolation¶

Type:enum in [‘NEAR’, ‘FAR’, ‘CW’, ‘CCW’], default ‘NEAR’
